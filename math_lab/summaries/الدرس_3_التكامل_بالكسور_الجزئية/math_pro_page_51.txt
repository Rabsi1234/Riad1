 1. القوانين باللاتيكس

 التكامل بالكسور الجزئية

التكامل بالكسور الجزئية هو تقنية تستخدم لحل التكاملات التي تحتوي على كسور رونية. يمكن تمثيل الكسور الجزئية على النحو التالي:

$$\frac{N(x)}{D(x)} = \frac{A_1}{x - r_1} + \frac{A_2}{x - r_2} + \cdots + \frac{A_n}{x - r_n}$$

حيث $N(x)$ هو البسط، $D(x)$ هو اللاسم، $r_i$ هي جذور اللاسم، و$A_i$ هي الثوابت.

 التكامل

التكامل هو عملية تعريفية تُستخدم لحساب المساحة تحت المنحني أو لحساب الكمية المتراكمة. يمكن تمثيل التكامل على النحو التالي:

$$\int_{a}^{b} f(x) dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i) \Delta x$$

 المتجهات

المتجهات هي كميات فيزيائية تحتوي على الحجم والاتجاه. يمكن تمثيل المتجهات على النحو التالي:

$$\vec{a} = (a_1, a_2, a_3)$$

 التوزيعات

التوزيعات هي كميات رياضية تحتوي على قيم احتمالية. يمكن تمثيل التوزيعات على النحو التالي:

$$P(X = x) = f(x)$$

 2. حل الأمثلة

 مثال 1

$$\int \frac{x^2 + 1}{x^3 - x} dx$$

يمكن حل هذا التكامل باستخدام الكسور الجزئية:

$$\frac{x^2 + 1}{x^3 - x} = \frac{A}{x} + \frac{B}{x^2} + \frac{C}{x - 1}$$

باستخدام طريقة التكامل بالكسور الجزئية، يمكن حل الثوابت $A$، $B$، و$C$:

$$A = 1, B = 1, C = 1$$

الآن يمكن حل التكامل:

$$\int \frac{x^2 + 1}{x^3 - x} dx = \int \frac{1}{x} dx + \int \frac{1}{x^2} dx + \int \frac{1}{x - 1} dx$$

$$= \lnx - \frac{1}{x} + \lnx - 1 + C$$

 مثال 2

$$\int \frac{x^3 + 2x^2 + 3x + 1}{x^4 - x^2} dx$$

يمكن حل هذا التكامل باستخدام الكسور الجزئية:

$$\frac{x^3 + 2x^2 + 3x + 1}{x^4 - x^2} = \frac{A}{x} + \frac{B}{x^2} + \frac{C}{x + 1} + \frac{D}{x - 1}$$

باستخدام طريقة التكامل بالكسور الجزئية، يمكن حل الثوابت $A$، $B$، $C$، و$D$:

$$A = 1, B = 1, C = 1, D = 1$$

الآن يمكن حل التكامل:

$$\int \frac{x^3 + 2x^2 + 3x + 1}{x^4 - x^2} dx = \int \frac{1}{x} dx + \int \frac{1}{x^2} dx + \int \frac{1}{x + 1} dx + \int \frac{1}{x - 1} dx$$

$$= \lnx - \frac{1}{x} + \lnx + 1 + \lnx - 1 + C$$

 3. أسئلة قدرات عليا

 سؤال 1

$$\int \frac{x^4 + 2x^3 + 3x^2 + 2x + 1}{x^5 - x^3} dx$$

 سؤال 2

$$\int \frac{x^5 + 3x^4 + 2x^3 + x^2 + 1}{x^6 - x^4} dx$$

 سؤال 3

$$\int \frac{x^6 + 2x^5 + 3x^4 + 2x^3 + x^2 + 1}{x^7 - x^5} dx$$

هذه الأسئلة تتطلب استخدام الكسور الجزئية لحل التكاملات. يمكن حلها باستخدام الطريقة نفسها التي استخدمت في الأمثلة السابقة.
